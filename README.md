# 📊 HR MIS Reports Analysis Dashboard - 2024

## 📌 Project Overview

The **HR MIS Reports Analysis Dashboard** is an interactive Microsoft Excel dashboard developed to monitor and analyze employee attrition trends. It provides HR professionals with real-time insights into workforce metrics such as headcount, attrition rates, tenure analysis, voluntary vs involuntary exits, and monthly attrition trends.

The dashboard enables users to filter data by **Phase, Process, Sub Process, Year, Month, Gender, and Tenure**, making it easy to perform detailed workforce analysis.

---

## 🚀 Features

- Interactive Excel Dashboard
- Dynamic Slicers for filtering data
- KPI Cards for key HR metrics
- Attrition Analysis by Tenure
- Voluntary vs Involuntary Attrition Analysis
- Monthly Attrition Trend Analysis
- Year-wise Reporting
- Gender-wise Analysis
- Easy-to-understand charts and visualizations

## 📈 Dashboard Components

Provides an overview of:

### 🔹 Voluntary vs Involuntary Attrition
Displays the distribution of:
- Voluntary Attrition
- Involuntary Attrition
- Total Employee Exits

### 🔹 Attrition by Tenure
Analyzes employee attrition based on tenure groups and gender to identify turnover patterns across different experience levels.

### 🔹 Monthly Attrition Trend
Shows monthly trends for:
- Voluntary Attrition %
- Involuntary Attrition %
- Total Attrition %

Users can analyze data using:
- Phase

- Year
- Month
- Gender
- Tenure

## 📊 Business Insights

- The organization has a total **Head Count of 204 employees**.
- The **overall attrition rate is 14.2%**, while the **YTD attrition rate is 13.3%**.
- The **annualized attrition rate stands at 19.9%**, highlighting the projected yearly employee turnover.
- Voluntary attrition contributes significantly more than involuntary attrition.
- Employees with **2–3 years of tenure** show the highest attrition percentage, indicating a critical retention period.
- Monthly trend analysis helps identify fluctuations in employee turnover and supports proactive workforce planning.
- Interactive filters allow HR teams to perform detailed analysis by department, process, gender, tenure, and reporting period.
